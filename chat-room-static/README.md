Static G+ Chat Room
================================

## chat-room-static

Representa o conteúdo estático do projeto G+ Chat Room. 

## Specifications

* Front-End (HTML/JavaScript)
* AngularJS

## Instalar e Executar

``` bash
$ git clone git@bitbucket.org:pflima92/chat-room-poc.git
$ cd chat-room-static
$ npm install
$ bower install
$ gulp serve
```

Sua aplicação deverá ser iniciada no endereço: http://localhost:3000
